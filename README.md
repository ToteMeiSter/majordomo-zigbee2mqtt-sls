# majordomo-zigbee2mqtt-sls

Модуль интеграции **Zigbee-устройств** в систему [MajorDoMo](https://mjdm.ru) через MQTT,
адаптированный для работы со шлюзами **[SLS](https://slsys.io)** (а также с классическим
[zigbee2mqtt](https://www.zigbee2mqtt.io)).

## Возможности

- Автообнаружение Zigbee-устройств и их параметров из MQTT-топиков шлюза.
- Управление устройствами (вкл/выкл, яркость, цвет, уставки и пр.) через `/set`.
- Поддержка нескольких шлюзов одновременно (мультишлюз).
- **Расширенная база моделей** (маппинг `zigbeeModel`/`fingerprint` → модель/производитель/описание)
  на основе данных проекта [zigbee-herdsman-converters](https://github.com/Koenkk/zigbee-herdsman-converters).
- **Точный резолв generic-моделей** (например `TS0601`) по `manufacturerName` — корректные
  тип устройства и картинка для разных устройств с одинаковым `modelID`.
- **Ленивая подгрузка картинок плиток**: при первом появлении устройства картинка
  автоматически скачивается с zigbee2mqtt.io и кэшируется локально.
- Группы, привязки (bind), карта сети, журнал.

## Установка

1. Скопировать содержимое в каталог MajorDoMo (`modules/`, `templates/`, `scripts/`).
2. Установить модуль через интерфейс MajorDoMo (создаст необходимые таблицы БД).
3. В настройках модуля указать параметры MQTT-брокера и путь подписки
   (для SLS — базовые топики шлюзов, например `gw1/#,gw2/#`).

## Требования

- MajorDoMo, PHP 7.4+ (проверено на PHP 8.3).
- MQTT-брокер (Mosquitto).
- Zigbee-шлюз: SLS или zigbee2mqtt.
